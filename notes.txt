Requirements:
python version 3.7
pip 18.0
CV2 3.4.2


1. Install open cv

$ pip install opencv-python

if error:

$ pip install opencv-contrib-python

2. run face_D.py

$ python face_D.py

3. Train dataset

$ python faceR.py

if error occurs: $ pip install image 

Then, $ pip install opencv-contrib-python

4. Recognition

$ python faceDetector.py
